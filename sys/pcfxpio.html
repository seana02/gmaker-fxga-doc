Name

PCFXPIO.SYS: PC-FXGA Parallel Interface Driver

Syntax

Put the command "DEVICE = PCFXPIP.SYS" into CONFIG.SYS. It is automatically written on install.

Functionality

Driver to provide the parallel interface between the PC-FXGA and PC-98. Uses the IOCTL command, an MS-DOS function, for data input/output. Provides functionality for input/outputting memory required for applications other than the debugger, as well as program execution and termination, etc.
Required to execute applications that use the PC-FXGA parallel interface. (AGDDISP.EXE, BMP2AID.EXE, AIDMERGE.EXE)
