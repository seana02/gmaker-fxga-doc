Chapter 1: Introduction

This library contains basic functions to directly control HuC6273. Most functions are written with speed and simplicity in mind, so there are no checks on passed in parameters. Please be careful that you don't set out-of-bounds values.

1.1 Functions and Macros

In general, function names are of the format FarlXxxxXxxx() and macro names are of the format FARL_XXXXXX
Segments are defined as follows. Please set them when linking.
#pragma defseg (-C, farl_code)
#pragma defseg (-D, farl_data)
#pragmadefseg(-Z,farl_zero)
#pragma defseg (-B, farl_bss)

Chapter 2: Farl Function Summary

2.1 Basic Functions

Basic functions used to access HuC6273

2.1.1 Initialize HuC6273

Name: FarlInit

Format:
#include <farl.h>
int FarlInit()


Parameters: None

Explanation:

Required when using FARL. Clears the display buffer, and Z buffer, and initializes the registers. Please see Chapter 3 for information about Farl register initialization.

Note: FarlGetConf() can be used without calling FarlInit()

Return Value:
 0: HuC6273 exists
-1: HuC6273 does not exist.

2.1.2 HuC6273 Reset

Name: FarlReset

Format:
#include <farl.h>
void FarlReset(int sw)


Parameters
sw:
  FARL_STATUS_CONTROL_HRST: Hard reset. All registers are reset to the value at power on.
  FARL_STATUS_CONTROL_SRST:: Soft reset. The FIFO is cleared and the HuC6273 is restarted. However, the contents of all registers are unchanged.

Explanation:

Resets or restarts the HuC6273

Return Value: None.

2.1.6 Reading TE Registers

Name: FarlGetTEReg

Format:
#include <farl.h>
unsigned short FarlGetTEReg(int regid, int sw)

Parameters:
regid: register's ID
sw:
  FARL_NOBLK: Sends the request, and returns without waiting for read back.
  FARL_BLK:: Waits for a read back and returns the value.

Explanation:

For information about the register's ID (regid), refer to the HuC6273's manual and farl.h.

If the sw parameter is set to FARL_NONBLK, the request is sent and does not wait for read back. Either poll the RBDONE bit of the Interrupt status register or read it from the Read back register after read back's completion is notified by an interrupt.

Return Value: (IF sw==FARL_BLK) The value from TE register

2.1.9 Writing to Texture Buffer (Pack Mode)

Name: FarlWriteTxt

Format:
#include <farl.h>
void FarlWriteTxt(unsigned short addr, unsigned short data)

Parameters:
addr: address from the start of the bank (0x0000 - 0xfffe) data: data to write in (16 bit)
Explanation:

Write data as 16 bit data value to the texture buffer using Pack mode (16 bit access mode). The address is specified using addr, which starts from the first address of the bank set in the CMT bank select register. The address must be even (word boundary). This function is primarily used to write 16 bit data like CMT and SCT.

Return Value: None.

Name: FarlWriteFifoW

2.1.18 Reading Overlay LUT Value

Name: FarlGetLut

Format:
#include <farl.h>
unsigned short FarlGetLut(int num, int sw)

Parameters:
num: LUT Number (1-3)
sw: FARL_NOBLK: Send only the request, and returns without waiting for the readback to conclude.
  FARL_BLK: Waits for readback to conclude before returning
Explanation:

Reads the value from the LUT specified by the LUT number (num).
If the sw parameter is set to FARL_NONBLK, only the request is sent, and it does not wait for readback to conclude. In this case, there is no return value. Read it from the Read back register after seeing the readback conclude either by polling the RBDONE bit of the Interrupt status register or the interrupter.

Return Value: (If sw==FARL_BLK) The value read from the LUT.

2.1.19 Setting Light Vector

Name: FarlSetLight

Format:
#include <farl.h>
void FarlSetLight(int num, unsigned short x, unsigned short y, unsigned short z)

Parameters:
num: Light number
x, y, z: Light's vector <1.0.15>
Explanation:

Sets the TE light vector specified by the light number (num) to the values x, y, z (<1.0.15> format). The light vector must be normalized to length 1. The light number (num) should be set to either FARL_SET_LIGHT1 or FARL_SET_LIGHT2 (refer to FarlSetLight1, FarlSetLight2).

Return Value: None.
