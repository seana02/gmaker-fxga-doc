Name

AGD2AGC: AGD→AGC File Converter

Syntax

AGD2AGC [Switch] AGD file name [AGC file name [ATT file name]]

Functionality

Converts AGD files into AGC files in command packet format. The first 4 bytes of the AGC file represent the size of the packet data, with the data starting after.

Switches

-x (-binary)
Outputs a binary file.

-a (-ascii)
Outputs a text file.
It is represented in hexadecimal starting with 0x.

-b (-byte)
When text file is outputted, it is outputted in bytes.

-h (-half)
When text file is outputted, it is outputted in half words (8 bits?).

-w (-word)
When text file is outputted, it is outputted in words (16 bits?).

-C
Outputs in C Array format.

-S
Outputs in Assembler's defined data format.
-banksel
Outputs the code choosing the texture bank before the commands drawing the polygons with the texture mapping.
-nobanksel Does not output the code choosing the texture bank before the commands drawing the polygons with the texture mapping.

-defcolor Outputs the code to set the default color before the commands drawing the polygons using default colors.

-nodefcolor Does not output the code to set the default color before the commands drawing the polygons using default colors.

-beef Outputs command with the termination word.

-nobeef
Outputs command without the termination word.

-noatt
When there is no ATT file or there is no entry corresponding to the geometry data’s ID, outputs assuming the UV coordinates are 0-255.
