Introduction

This sample program plays ADPCM files with the .adp extension.

It does not support Stereo ADPCM files, and files greater than 256 kilobytes in size are may not work as intended.

Execution

Run FXDB.EXE, and run the following commands:

y adpplay [CR] rg [CR]

Controls

When the program is run, the host machine will display

Input filename:

Specify the adp file that you want played. Adding + at the end of the file name will make it play 5 times in a row. Please utilize this for when you want to listen to a short sound consecutive times.

Compiling

After making changes to ADPPLAY.C, execute MK.BAT. Please make sure to specify the execution path of the compiler and linker.
